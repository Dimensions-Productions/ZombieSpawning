# Region Selection and Zombie Spawn Customizer
Allows easy region creation and enables you to set a limit to the amount of zombies that can spawn in said region, also has a interval and amount setting to customize zombie spawn speeds

## Guide: 

### Commands:

```
/edit 
```
Gives you a region creation wand
```
/editregion
```
Opens region edit menu for modifying Limit, Interval and amount of spawns
```
/deleteregion
```
Deletes region you are standing in

Default Limit, Interval and amount are modifyable in ZombieSpawning\chunks.yml
Default values are:
```
DefaultGlobalLimit: 5
DefaultGlobalInterval: 100
DefaultGlobalAmount: 1
```

### Note:

Region creation is 2D and Y level does not matter
